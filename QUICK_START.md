# ⚡ PlaceOPrep - QUICK START CARD

## 🚀 IN 3 STEPS:

### 1️⃣ FIRST TIME SETUP
```bash
# Make sure MongoDB is installed and running
mongod

# Double-click this file:
setup.bat

# Wait for it to complete (installs everything + seeds database)
```

### 2️⃣ START APPLICATION
```bash
# Double-click this file:
start.bat

# This starts:
# - Backend (http://localhost:4000)
# - Frontend (http://localhost:5173)
# - AI Engine (http://localhost:5000)
```

### 3️⃣ USE APPLICATION
```
1. Open: http://localhost:5173
2. Click "Sign up" 
3. Enter: Name, Email, Password
4. Click "Sign Up"
5. You're in! Start practicing!
```

## 🆘 QUICK FIX:

**Nothing works?**
1. Is MongoDB running? → Start `mongod`
2. Did you run setup.bat? → Run it once
3. Did you run start.bat? → It starts all services

**Can't see questions?**
→ Run: `cd backend && node seed.js`

**Can't login?**
→ Click "Sign up" first to create account
